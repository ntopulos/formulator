What is Formulator ?

It is the solution for the forms on the Web. Say goodbye to these painful implementations of forms, say hello to Formulator!

Formulator is a PHP library that fully handles forms. It takes care of the generation and the validation, it is complete and reliable without compromise on the flexibility: you can easily integrate it in any project and framework.

Define the elements of your form and their constraints, Formulator will take care of the rest!


$formulator = new Formulator();
$formulator->addElement('username');
$formulator->elements->root->username->addRule('alpha_numeric');
$formulator->addElement('message', 'Textarea');

if ($formulator->compute()) { }

echo $formulator->render(true);

Formulator will not only generate this final HTML form, it will also validate its content and allow you to easily access the data.

Looking for something more complex?


$formulator = new Formulator();
$formulator->addElement('email', 'InputEmail');

$formulator->addElement('birthday', 'InputDate');

// lets add a rule!
$formulator->addElement('finger_number', 'InputNumber');
$formulator->elements->root->finger_number->addRule('between:0,10');

$formulator->addElement('newsletter', 'Checkboxes');

$colors = array('Orange', 'Green', 'Brown');
$formulator->addElement('color', 'Radios');
$formulator->elements->root->color->options = $colors;

if ($formulator->compute()) {
    // some action
}

echo $formulator->render(true);
