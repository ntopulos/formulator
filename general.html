Features

General:

  • Fully functional forms and their validations with a very small amount of code
  • Constraints are automatically used in both the HTML elements and for the validation
  • The form's render is designed to be integrated within any template engine
  • Automatic forms generation based on MySQL tables or queries
  • Direct forms rendering with an integrated pseudo template engine

Installation

If instead of integrating Formulator in some project, you just want to try it on its own: ยง Try it.

The package utilizes Composer for installation.

Yet, as the development is in early stage, there is no Packagist package available. Instead, you can create a composer.json like this one:


{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/ntopulos/formulator"
        }
    ],
    "require": {
        "ntopulos/formulator": "dev-master"
    }
}

Then, run the following command in your terminal:


composer install
By now you should be able to use Formulator: 

// autoload
require_once 'vendor/autoload.php';

// call with the namespace
$formulator = new ntopulos\formulator\Formulator();

Try it

If you want to quickly try Formulator by yourself and test it, you should download the formulator-devkit.

Requirements

Formulator is a standalone library, basically you just need PHP.

Required:

  • PHP >= 5.4.16

Required for database assisted generation:

  • Mysqli or PDO
  • MySQL >= 5.1.37 for advanced generation

Recommended:

  • A template engine
