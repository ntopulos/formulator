Introduction

The functioning of Formulator can be divided in three steps:

  1. Definition of the elements and their rules.
  2. Computation
  3. Rendering

Simple form example


/* Step 1
 * creation of the form and definition of all elements */
$formulator = new Formulator();

$formulator->addElement('username');
$formulator->elements->root->username->addRule('alpha_numeric');

$formulator->addElement('email', 'InputEmail');


/* Step 2 
 * computation */
if ($formulator->validate()) {
    // if the form is successfuly validated
    // some action can be performed (e.g.: data use and redirect)
}


/* Step 3
 * rendering */
echo $formulator->renderTemplate();

More detailed information about the above steps can be found in the related sections of this documentation.

Notice that $formulator->validate() is necessary to populate the form with the submitted data. The submitted() method is also available, which allows to check if the form has been submitted without any validation or population.

Advanced form examples


/* Step 1
 * creation of the form and definition of all elements */
$formulator = new Formulator();


// defining some options
$formulator->auto_placeholders = true;          // show HTML placeholders
$formulator->human_readable_labels = false;     // labels will be equal to the names


// manually adding an element
$formulator->addElement('username');
$formulator->elements->root->username->addRule('alpha_numeric');


// adding elements from a Mysqli query
$result = $mysqli->query($query);               // select with multiple results

$options = array(
'addresses'   => array('type'        => 'InputTextarea',
                       'rules'       => array('required'),
                       'attributes'  => array('style' => 'color: #ad1700')
             ),
 'birth_date' => array('ignore' => true)
 );

$formulator->addEmptyElementsFromQueryObject($result, $options);



/* Step 2 
 * computation */
if ($formulator->validate()) {
    // if the form is successfuly validated
    // some action can be performed (e.g.: data use and redirect)
}


/* Step 3
 * rendering */
$render =  $formulator->render();   // array to send to your custom template

Loading data to the form

When the elements are added using the assisted generation, the values are automatically loaded from the source.

When manually adding an element, you can define its default value like this:


$default_value = 'ftp://';
addElement('ftp_address', 'InputText', $default_value)

Retrieving data from the form

In this early development stage there is not yet any specific method to address this task. However you can easily directly access the POST or GET array after the validation.
